# WalletWise

## Giới thiệu
WalletWise là một ứng dụng quản lý chi tiêu cá nhân, giúp bạn theo dõi thu nhập và chi tiêu, phân tích thói quen chi tiêu, và hiển thị dữ liệu dưới dạng biểu đồ trực quan.

## Các tính năng chính
- **Thêm giao dịch**: Nhập số tiền, mô tả, loại giao dịch, danh mục, và ngày thực hiện giao dịch.
- **Sửa giao dịch**: Chỉnh sửa thông tin giao dịch một cách dễ dàng.
- **Xóa giao dịch**: Xóa giao dịch không cần thiết với hiệu ứng animation.
- **Xem danh sách giao dịch**: Hiển thị tất cả các giao dịch theo thời gian, hỗ trợ phân trang.
- **Tìm kiếm giao dịch**: Tìm kiếm theo từ khóa hoặc danh mục.
- **Lọc giao dịch nâng cao**: Lọc theo loại giao dịch (Thu/Chi) hoặc khoảng thời gian (Hôm nay, Tuần này, Tháng này).
- **Thống kê chi tiêu**: Hiển thị tổng thu/chi và phân tích theo danh mục với biểu đồ dạng cột hoặc tròn.
- **Sao lưu và khôi phục dữ liệu**: Lưu trữ dữ liệu cục bộ và khôi phục khi cần thiết.

## Công nghệ sử dụng
- **Frontend**: React Native (Expo)
- **Quản lý trạng thái**: Redux Toolkit
- **Cơ sở dữ liệu**: MongoDB Realm (kết nối với MongoDB Atlas)
- **Styling**: Tailwind CSS
- **Biểu đồ**: Victory Native
- **Lưu trữ cục bộ**: AsyncStorage

## Cách cài đặt
1. Clone repository:
   ```bash
   git clone https://github.com/SilcoSudo/WalletWise.git
   cd WalletWise
   ```

2. Cài đặt các gói cần thiết:
   ```bash
   yarn install
   ```

3. Khởi chạy ứng dụng:
   ```bash
   yarn start
   ```
